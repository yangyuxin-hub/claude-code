# Claude Code 课程社区建设项目

## 项目定位

公众号 + 课程变现。面向只用过豆包/Kimi/千问的国内零基础用户，帮他们从"AI 工具用户"成长为"真正会用 AI 的人"。

**策略顺序**：先做公众号积累受众 → 再预售课程变现 → 再上知识星球

## 本文件夹内容说明

`Claude Code课程体系.md` 是核心战略文档，包含：思想体系、文章规划、课程结构、变现矩阵、发布节奏。**读它就能了解整个项目全貌。**

> 其他背景（课程录制细节、社群规划等）的对话记录在 claude.ai 官网。

## 文件结构约定

每篇文章独立文件夹，包含：稿件.md / outline.md / prompts/ / 图片文件

```
{文章标题}/
├── {文章标题}.md
├── outline.md
├── prompts/00-cover.md, 01-xxx.md ...
├── 00-cover.png（2.35:1 封面）
└── 01~NN.png（正文图 16:9）
```

配图生成方式见父目录 `CLAUDE.md`（baoyu-danger-gemini-web，并行生成）。
