# 我的原创核心思想体系

## 一、人与 AI 协作的四个进化阶段

|阶段|AI 角色|人的角色|代表工具|
|---|---|---|---|
|第一阶段|对话工具|操作者|豆包、Kimi、千问、Claude.ai 网页版|
|第二阶段|执行 Agent|监督者|Claude Code、Cursor|
|第三阶段|战略伙伴|决策者|Claude + Projects + Memory|
|第四阶段|自主运行|所有者|未来 AI 员工/公司|

大多数人停在第一阶段。课程帮人跨越到第二、第三阶段。星球和高阶课覆盖第四阶段。

## 二、实习生思维 vs 工具思维

**工具思维**：你告诉 AI 做什么

- 你需要对整个工作流程清楚，知道怎么做，只是把执行过程交给 AI
- AI 是提效工具，你的思维带宽是瓶颈

**实习生思维**：你告诉 AI 要什么结果

- 你化身老板，不需要对业务怎么做有深入理解
- 只需要把背景和目标说清楚，AI 自行规划、决策、执行
- 你的任务从"提效"变成"怎么把需求信息化交代清楚"

给 AI 的三要素：**背景 + 目标 + 边界**

## 三、潘多拉魔盒：行业经验贬值

实习生思维打开了潘多拉魔盒——

- 旧世界护城河：行业经验，知道"怎么做"
- 新世界：AI 把流程知识变成了公共品，在完全不了解的领域 AI 也能帮你完成
- 行业经验积累成了笑话
- **执行力贬值，判断力升值**

新时代真正稀缺的四种能力：

1. 知道要什么（目标感、品味、判断力）
2. 看得出好坏（AI 给5个方案，能判断哪个对）
3. 信息结构化（把模糊想法变成 AI 能执行的清晰指令）
4. 提对问题（问错方向 AI 再强也没用）

## 四、AI 击碎神话

**已经击碎的**

- 创意是人类独有的 → AI 写作/绘画/作曲已超过大多数人类平均水平
- 编程需要多年学习 → 描述需求，AI 直接出代码
- 专业知识是护城河 → AI 通过了律师、医师、注册会计师资格考试
- 经验需要时间积累 → AI 调用的是人类所有公开知识的总和

**正在击碎的**

- 代码：Anthropic 内部超过50%的代码已由 AI 生成；外包程序员、初级工程师、测试工程师被冲击最惨
- 金融：高盛2000年600个交易员，现在只剩2个；摩根大通 AI 分析合同，律师团队600小时的工作几秒完成

**可预见会击碎的**

- 科学发现、医疗判断、教育传授、中层管理决策

## 五、海啸意象与迭代速度

Anthropic 创始人 Dario Amodei：人们面对 AI 就像站在海滩上，远处海啸已经可见，但大家还在继续捡贝壳。

迭代速度：按年 → 按月 → 按周 → 按天（人类历史上从未有过任何技术以这种速度演进）

时间锚：你还能想象三年前没有 AI 时怎么学习、写作、完成工作吗？ 接下来三年的变化幅度，会是过去三年的十倍。

顶尖 AI 公司对 AGI 到来的预期越来越近，而这一切离普通人似乎还很远——这个认知差本身就是最大的风险。

# AI 记忆管理三条法则

**法则一：项目隔离** 不同项目建独立文件夹 + 独立 CLAUDE.md，绝不混用。

**法则二：结构化沉淀** 每次有价值的对话，提炼成结构化 Markdown 存入 Obsidian 或项目文件夹。 对话是消耗品，文档才是资产。

**法则三：规则可迁移** 所有重要规则写进 CLAUDE.md，换电脑/换项目直接复制，无缝继续。

# 入门必知的4个概念（面向零基础的解释方式）

|概念|给零基础的解释|
|---|---|
|上下文长度|AI 的短期记忆，对话太长会忘掉前面说的，导致越聊越跑偏|
|Token 消耗|每句话都在花钱，废话多=烧钱快，效果还差|
|CLAUDE.md|给 AI 的入职手册，写清楚项目背景和规则，每次启动自动读取|
|模型选择|Opus/Sonnet/Haiku 各有适合场景，用错了钱白花|

# 进阶知识点

- 上下文管理：控制对话长度、何时开新对话、/compact 压缩
- CLAUDE.md 深度写法：项目规范、代码风格、禁止行为、角色设定
- Token 成本控制：估算任务成本、省 token 的写法、批量任务规划
- 模型选择策略：不同场景用不同模型

- LanceDB：向量数据库，给 AI 加长期记忆
- Obsidian 联动：本地知识库 + 自动整理笔记 + 双向链接
- Superpowers 等扩展工具：给 Claude Code 加能力
- 小龙虾（OpenClaw）：Claude Code 的国内套壳版本，底层能力完全来自 Claude Code

# 写作风格规则

- 面向只用过豆包/Kimi/千问的普通用户，不假设读者知道 Claude
- 不用技术术语，用生活化比喻
    - 不说「Agent Loop」，说「AI 自己想步骤自己执行」
    - 不说「Context Window」，说「AI 的记忆容量」
    - 不说「Token」，说「AI 处理文字要花钱，废话越多越贵」
- 每篇必须有"当头棒喝"的冲击感，读者看完有"我原来想错了"的感觉
- 每篇结尾必须有评论引导，冷启动期互动率很重要
- 每篇结尾预告下一篇，制造追更习惯
- 第一人称为主，真实感比权威感更重要
- 课程脚本每节必须有「当堂成果」，让学员能立刻截图发朋友圈
