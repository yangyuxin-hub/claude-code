# 策略（月度迭代）

## 当前最优打法

### 求职策略

- **投递方向**：AI 应用开发（大厂）+ 量化，已筛 7-8 个岗位
- **项目积累**：正在学两个金融 Agent 项目，同步做自己的 agent 项目
- **公众号价值**：建立技术视角的个人影响力——让潜在雇主看到你怎么思考 AI

### 内容策略

- **主题**：Anthropic 相关——Claude 系列模型、Claude Code、Anthropic 产品理念、AI 前沿
- **首要目标**：写出自己满意的内容，传播性是加分项不是前提
- **发布节奏**：不死板，有感觉就写，热点出现就优先发
- **传播性自检**：发前问自己「这篇我自己满意吗？」
- **标题逻辑**：具体场景 + 具体信息，避免泛泛的"AI 来了"式煽情

### 小红书策略

**平台分工**：XHS 是触达新读者的入口，公众号是深度内容主体

**内容分工**：
- XHS：热点解读为主（短平快，500字内），偶尔发公众号精华版压缩
- 公众号：完整长文，Anthropic 深度解读

**格式**：长文本为主（制作成本低，算法友好）

### 侧面观察的方向

AI 使用课程（CoWorker / Claude Code / 可外包可复制工作）：市场存在（传统企业主有 AI 转型需求，技术人与业务端存在断层），暂不是工作重心，持续观察。

### 系统策略

- **知识沉淀**：做完一件事就记录，不靠记忆，靠文件
- **AI 协作**：关键文件都放在 CLAUDE.md @引用链里，新会话不重复对齐
