# Anthropic API 更新日志精选（2025年至今）

> 完整日志：https://platform.claude.com/docs/en/release-notes/overview
> 整理时间：2026-03-27

---

## 2026 年

### 2026-03-18
- Models API 新增模型能力字段：`max_input_tokens`、`max_tokens`、`capabilities` 对象

### 2026-03-16
- Extended thinking 新增 `display` 字段：可省略思考内容以加快 streaming（`thinking.display: "omitted"`）

### 2026-03-13
- **1M token 上下文窗口正式 GA**（Opus 4.6 / Sonnet 4.6），200k 以上自动生效，无需 beta header
- 每请求图片/PDF 上限从 100 提升至 600 页

### 2026-02-19
- **自动缓存上线**：Messages API 支持 `cache_control` 字段，无需手动管理缓存断点

### 2026-02-17
- **Claude Sonnet 4.6 发布**：支持 extended thinking + 1M token 上下文（beta）
- 代码执行工具与 web search/fetch 配合使用时免费
- Web search 和 programmatic tool calling 正式 GA（无需 beta header）

### 2026-02-07
- Opus 4.6 Fast Mode 研究预览：最高 2.5x 速度，需加入 waitlist

### 2026-02-05
- **Claude Opus 4.6 发布**：最强推理能力，推荐用 adaptive thinking
- Compaction API beta：服务端上下文压缩，支持无限长对话
- 数据驻留控制：`inference_geo` 参数指定推理地区
