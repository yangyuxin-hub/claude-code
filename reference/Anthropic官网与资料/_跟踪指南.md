# Anthropic 信息跟踪指南

| 来源 | 地址 | 更新频率 | 内容类型 |
|------|------|----------|----------|
| Anthropic 官方博客/新闻 | https://www.anthropic.com/news | 每周1-3篇 | 产品发布、公司动态、研究 |
| API 更新日志 | https://platform.claude.com/docs/en/release-notes/overview | 每周1-2次 | API 功能变更 |
| Claude Code CHANGELOG | https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md | 几乎每天 | CLI 版本更新 |
| Claude Code GitHub | https://github.com/anthropics/claude-code | 实时 | Issues、PR、讨论 |
| Claude 帮助中心更新 | https://support.claude.com/en/articles/12138966-release-notes | 每周 | Claude App 产品更新 |

## 建议跟踪节奏

- **每天**：扫一眼 Claude Code GitHub（有无新版本）
- **每周一次**：看 Anthropic News 有无新文章，API 更新日志
- **有重大发布时**：立即存档到对应目录，记录要点

## 文件命名规范

```
YYYY-MM-DD_来源_标题简述.md
```

例如：
- `2026-03-11_官方博客_Anthropic-Institute-成立.md`
- `2026-02-17_产品更新_Claude-Sonnet-4.6发布.md`
- `2026-03-27_ClaudeCode_v2.1.84更新.md`

## 哪些内容值得存档

✅ 值得存档：
- 新模型发布（含性能对比数据）
- Claude Code 重大功能更新
- 创始人访谈中关于 AI 趋势/就业的观点
- Anthropic 对 AI 安全/商业的战略表态
- 经济研究报告（如 Anthropic Economic Index）

❌ 不必存档：
- 人事任命、办公室开设等纯公司新闻
- 合作协议类公告（无实质技术内容）
- API 小版本 bug fix
