# 案例：公众号内容自动发布

> 行业：自媒体 / 内容创作｜完成时间：2026-03

## 客户背景

- 个人公众号创作者
- 原来流程：手动排版 → 上传图片 → 填写摘要 → 发布，每篇耗时 30-60 分钟
- 核心痛点：重复性操作多，容易出错，图片上传繁琐

## 解决方案

- 工具：Claude Code + baoyu-post-to-wechat + 微信公众号 API
- 流程：
  1. 本地写好 `.md` 文件
  2. 一条命令自动压缩图片 + 上传 + 生成摘要 + 写入草稿箱
  3. 登录微信后台确认发布
- 关键技术点：微信素材 API 上传图片，Markdown 转微信兼容 HTML

## 效果

- 发布时间从 30-60 分钟压缩到 2-3 分钟
- 图片上传错误率降为 0
- 可批量处理多篇文章

## 可复制性

- 适用：所有有公众号的企业/个人（品牌号、企业服务号、个人号）
- 复制难度：低
- 需要调整：微信 AppID / AppSecret 配置

## 经验教训

- 微信图片素材有大小限制，需要提前压缩
- 草稿箱和直接发布是两个不同 API，注意区分
- 摘要自动生成效果好，节省时间明显
